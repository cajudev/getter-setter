Leia a documentação completa em: https://cajudev.readthedocs.io/pt_BR/latest/getter-setter

## Características

-   Segue os padrões PSR-1, PSR-2 e PSR-4
-   Testes unitários com PHPUnit

## Instalação

``composer require cajudev/getter-setter "^0"``
